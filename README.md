# Student Management System

A command-line application built in Python to manage student records.

## Features
- Add new students with marks for 5 subjects
- View all students with complete report cards
- Search students by name
- Find class topper automatically
- Grade calculation (A, B, C, D)

## Technologies Used
- Python 3

## What I Learned
- Functions
- Dictionaries and nested dictionaries
- Lists
- Loops and conditionals
- String manipulation
